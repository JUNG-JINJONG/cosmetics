# 코스맥스 플러스 분석 기능 목록 (Function List - 02)

코스맥스 플러스(COSMAX PLUS)의 온라인 화장품 개발 플랫폼 분석 결과 도출된 핵심 기능입니다.

## 1. ODM/OBM/OEM 개발 워크플로우
*   **원스톱 개발 의뢰**: 회원가입 후 단계별 설문을 통해 브랜드 컨셉부터 제형, 패키징까지 한 번에 의뢰 가능.
*   **실시간 진행 현황 트래킹**: [의뢰 접수 -> 검토 -> 견적/계약 -> 발주 -> 생산 -> 출고] 전 과정을 실시간으로 모니터링.
*   **유연한 생산 대응 (MOQ 0)**: 인플루언서 및 소규모 인디 브랜드를 겨냥한 유연한 최소 주문 수량 기능.

## 2. 특화 라이브러리 (Assets Library)
*   **COSMAX PKG (용기 검색)**: 코스맥스 전용 용기(MOLD) 및 부자재를 제형 적합성이나 용량별로 검색 및 선정.
*   **R&D 특허 라이브러리**: 코스맥스 보유 특허 성분, 독점 원료, 검증된 베이스 제형 카탈로그 공유.
*   **트렌드 인사이트**: 글로벌 뷰티 시장 데이터 및 소비자 동향 리포트 상시 제공.

## 3. 사용자 맞춤형 서비스
*   **상세 문진 자동화**: 제형 성격(향, 타겟 단가, 수출 국가, 용량 등)을 세부적으로 설정하여 연구원에게 정밀한 데이터 전달.
*   **파트너십 관리**: 국내외 의뢰인(Brand Client) 및 원료/부자재 공급사(Partner) 구분 관리.
*   **챗봇 상담**: 실시간 온라인 기술 상담 및 개발 가이드 제공.
